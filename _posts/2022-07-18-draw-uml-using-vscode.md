## Requirement

```
Java should be installed on your machine
```

## Commands

- Search PlantUML from Marketplace in VSCode

- Install PlantUML

- Create a file with the ".plantuml" extension

```
ex) squence.plantuml
```

- Start with "@startuml"

- Write down some script code

- End with "@enduml"

- See the corresponding UML by typing "Alt + D"
